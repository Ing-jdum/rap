License

RAP (ROS Agent Planner) is an open-source research project designed for integrating generative AI into robotic systems using ROS2.

Credits

This project is built upon several amazing open-source technologies and frameworks:

  • ROS2 (Robot Operating System 2) - The foundation of this project, providing the middleware for robot control.
  • PlanSys2 - The planning system used for high-level task execution.
  • Nav2 (Navigation Stack 2) - Used for autonomous robot navigation.
  • LangChain - Provides AI-based language understanding and planning capabilities.
